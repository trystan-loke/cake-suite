# Cake Suite Frontend

Frontend for Cake Suite, a SaaS for cakers. It uses:

- Vue 3
- Vite 4
- Firebase Auth (email/password)

## Dev

- Requires Node 14+ (this project pins Vite 4 for broad compatibility)

### Run

```sh
npm install
npm run dev
```

Open http://localhost:5173.

### Build

```sh
npm run build
npm run preview
```
